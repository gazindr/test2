Управление между элементами происходит путем нажатия кнопок WASD и стрелок.

Активация кнопки происходит путем нажатия кнопки Enter (или ОК)

Для перехода на главный экран необходимо нажать на кнопку ОК. 

Переход на предыдущий экран происходит путем нажатия кнопки Escape (Back). Переход на предыдущий экран доступен только из экранов Настроек и Кредитс.

На экране настроек демонстрация работы управления со слайдерами, кнопкой и чекбокса.
